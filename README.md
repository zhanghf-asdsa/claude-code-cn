# Claude Code 中文文档

> Claude Code 官方文档的中文翻译版本

## 📖 简介

这是一个纯静态网站，为中文开发者提供 Claude Code 的完整中文文档。Claude Code 是 Anthropic 官方的 AI 辅助开发命令行工具。

## 🌐 访问网站

- **在线文档**：https://zhanghf-asdsa.github.io/claude-code-cn/
- **官方文档**：https://docs.anthropic.com/claude/docs/overview

## ✨ 特性

- 📚 **完整文档** - 涵盖快速开始、核心概念、命令参考、技能开发和 MCP 集成
- 🎨 **极简设计** - 清爽的灰白配色，专为中文阅读优化
- 📱 **响应式** - 完美适配桌面和移动端
- 🔍 **全文搜索** - 纯前端搜索，快速定位内容
- 🎯 **代码高亮** - 支持多种编程语言的语法高亮
- ⚡ **快速加载** - 纯静态 HTML，无构建依赖

## 📁 文档结构

```
claude-code-cn/
├── index.html              # 首页
├── docs/
│   ├── getting-started/    # 快速开始
│   ├── concepts/           # 核心概念
│   ├── commands/           # 命令参考
│   ├── skills/             # 技能开发
│   └── mcp/                # MCP 集成
└── assets/
    ├── css/                # 样式文件
    └── js/                 # 脚本文件
```

## 🛠️ 本地运行

### 使用 Python

```bash
cd claude-code-cn
python -m http.server 8000
```

然后访问 http://localhost:8000

### 使用 Node.js

```bash
cd claude-code-cn
npx serve .
```
