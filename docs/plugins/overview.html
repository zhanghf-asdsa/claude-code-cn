插件概述

Claude Code 插件是一种强大的扩展方式，允许您自定义和增强 Claude Code 的功能。通过插件，您可以添加自定义命令、创建专门的代理、实现技能系统，并集成外部工具。

为什么使用插件？

  • 自定义命令 - 创建团队专用的斜杠命令，自动化常见工作流程
  • 专门代理 - 针对特定任务（如代码审查、文档生成）的 AI 代理
  • 技能系统 - 为 Claude 添加特定领域的知识和能力
  • 事件驱动 - 通过钩子在特定事件发生时执行自定义逻辑
  • 工具集成 - 通过 MCP 连接外部服务和 API
  • 团队共享 - 通过插件市场与团队共享自定义功能

插件组件

一个 Claude Code 插件可以包含以下组件：

1. 斜杠命令（Commands）

通过简单的斜杠命令触发复杂的任务。例如：

/review          # 代码审查
/deploy          # 部署应用
/test            # 运行测试

2. 代理（Agents）

具有特定专业知识和行为的自主 AI 代理。例如：

  • 代码审查专家
  • API 文档编写者
  • 测试生成器

3. 技能（Skills）

为 Claude 添加特定领域的专业知识。技能会根据上下文自动激活。

4. 钩子（Hooks）

在特定事件发生时执行自定义脚本：

  • PreToolUse - 工具使用前
  • PostToolUse - 工具使用后
  • SessionStart - 会话开始时
  • Stop - 会话结束时
  • UserPromptSubmit - 用户提交提示时

5. MCP 服务器

集成外部工具和服务，支持多种通信方式：

  • stdio - 本地进程通信
  • SSE - 服务器发送事件（支持 OAuth）
  • HTTP - RESTful API 调用
  • WebSocket - 实时双向通信

插件目录结构

plugin-name/
├── .claude-plugin/
│   └── plugin.json          # 插件清单（必需）
├── commands/                 # 斜杠命令（.md 文件）
│   └── my-command.md
├── agents/                   # 代理定义（.md 文件）
│   └── my-agent.md
├── skills/                   # 技能模块
│   └── my-skill/
│       └── SKILL.md         # 技能定义
├── hooks/                   # 事件处理器
│   ├── hooks.json           # 钩子配置
│   └── my-script.py         # 钩子脚本
├── .mcp.json                # MCP 服务器配置
└── README.md                # 插件文档

安装和使用插件

从插件市场安装

# 列出可用插件
claude

# 在 Claude Code 中安装插件
> /plugin install superpowers

# 查看已安装的插件
> /plugin list

本地开发插件

在项目目录创建 .claude/ 文件夹并添加插件配置：

{
  "plugins": [
    "path/to/local-plugin"
  ]
}

插件清单（plugin.json）

每个插件必须有一个 plugin.json 清单文件：

{
  "name": "my-plugin",
  "version": "1.0.0",
  "description": "我的 Claude Code 插件",
  "author": "Your Name",
  "license": "MIT"
}
