内置命令

Claude Code 提供了一些内置命令来帮助您控制会话和行为。

基础命令

/help

显示帮助信息，列出所有可用命令。

/help

/clear

清除当前会话的对话历史，开始新的对话。这不会影响已修改的文件。

/clear

/exit

退出当前的 Claude Code 会话。

/exit

也可以使用 Ctrl+C 来退出。

文件操作命令

/read

让 Claude 读取指定文件的内容。

/read <文件路径>

示例：

/read src/app.js

/write

让 Claude 写入或创建文件。

/write <文件路径>

然后 Claude 会询问您要写入的内容。

项目管理命令

/git

执行 Git 命令。

/git <git 命令>

示例：

/git status
/git diff
/git commit -m "feat: add new feature"

/run

运行脚本或命令。

/run <命令>

示例：

/run npm test
/run python script.py

配置命令

/config

查看或修改 Claude Code 配置。

/config                    # 查看所有配置
/config <key>              # 查看特定配置
/config <key> <value>       # 设置配置

/model

切换 AI 模型。

/model                    # 查看当前模型
/model <模型名称>          # 切换模型

快捷键

快捷键 功能
Ctrl+C 退出会话
Ctrl+D 退出会话（同 Ctrl+C）
Ctrl+L 清屏（不清除对话历史）
↑ / ↓ 浏览命令历史

命令别名

一些命令有简写形式：

完整命令 简写
/exit /x
/clear /c
/help /h, ?

最佳实践

  • 使用 /clear - 当切换到新任务时清除对话历史
  • 定期提交 - 使用 /git commit 定期保存更改
  • 测试运行 - 使用 /run 测试您的更改
  • 查看帮助 - 忘记命令时使用 /help
