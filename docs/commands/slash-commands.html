斜杠命令

斜杠命令是以 / 开头的特殊命令，用于快速执行特定操作或切换模式。

命令分类

会话控制

命令 说明
/help 显示帮助信息
/clear 清除对话历史
/exit 退出会话
/reset 重置会话，清除所有上下文

文件操作

命令 说明
/read <path> 读取文件内容
/write <path> 写入文件内容
/edit <path> 编辑文件
/delete <path> 删除文件

项目操作

命令 说明
/git <cmd> 执行 Git 命令
/run <cmd> 运行 shell 命令
/test 运行项目测试
/build 构建项目

模式切换

命令 说明
/code 进入代码模式
/chat 进入聊天模式
/plan 进入规划模式

高级用法

命令链

可以用 &;> 连接多个命令：

/read src/app.js >> /edit src/app.js >> /test

命令参数

许多命令支持额外的参数：

/read src/app.js --line 1-50
/git commit --amend

命令别名

可以创建自定义命令别名：

/alias save "/git add . >> /git commit"

Tab 自动补全

输入命令时按 Tab 键可以自动补全：

/rea<Tab>        # 自动补全为 /read
/git ch<Tab>      # 自动补全为 /git checkout

命令历史

使用 ↑ 和 ↓ 键浏览之前执行的命令。

配置自定义命令

在项目根目录创建 .claude/commands 目录，可以添加自定义命令：

.claude/
└── commands/
    ├── deploy.md
    └── test.md

然后可以使用 /deploy 或 /test 调用这些自定义命令。

最佳实践

  • 使用 Tab 补全 - 提高输入效率
  • 创建别名 - 简化常用操作
  • 组合命令 - 使用命令链执行复杂任务
  • 查看帮助 - 不确定时使用 /help
