安装 Claude Code

Claude Code 是 Anthropic 官方提供的 AI 辅助开发命令行工具。本指南将帮助您完成安装。

系统要求

  • Node.js 18 或更高版本
  • npm、yarn 或 pnpm
  • macOS、Linux 或 Windows (WSL)

安装步骤

使用 npm 安装

npm install -g @anthropic-ai/claude-code

使用 yarn 安装

yarn global add @anthropic-ai/claude-code

使用 pnpm 安装

pnpm add -g @anthropic-ai/claude-code

验证安装

安装完成后，运行以下命令验证：

claude --version

如果显示版本号，说明安装成功！
