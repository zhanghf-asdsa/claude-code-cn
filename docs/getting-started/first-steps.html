第一步

恭喜您安装了 Claude Code！让我们开始第一次对话。

启动 Claude Code

在您的项目目录中运行：

claude

您的第一个请求

尝试让 Claude Code 帮您完成一个简单任务：

> 帮我创建一个 Hello World 函数

常用命令

命令 说明
/help 显示帮助信息
/clear 清除对话历史
Ctrl+C 退出 Claude Code
